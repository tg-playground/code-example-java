# grpc-demo

## Generating Protocol Buffer Code with Gradle

```shell
./gradlew generateProto
```

The output directory of generated source code: `app/build/generated/sources/proto`

Reference: [protobuf-gradle-plugin - GitHub](https://github.com/google/protobuf-gradle-plugin)

## gRPC Communication Patterns

unary RPC (simple RPC), server-side streaming, client-side streaming, and bidirectional streaming.
